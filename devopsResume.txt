* Cycle de vie d'un logiciel 
* l'agile

Definition :

Devops est l'union de : 
-personnes / - processus / - produits , pour permette une livraison continbe, de valeur, à nos final users


=> Dev   U     Ops

=> includes : planification agile,CI (Continuous Integration) , & CD (Continuous Delivery)

=> Vise à céer une culture et un environnement dans lesquels la conception, les tests et la diffusion de logiciels peuvent être réalisés rapidement, fréquemment et efficacement.
DevOps n'est pas seulement une méthodologie, c'est une véritable philosopie de travail.



* CI : Continuous Integration

Fusion of code and unit test ==> detect bugs earlier


* CD : Continuous Delivery : use of pipelines
